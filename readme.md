# CGPA Calculator

A simple and customizable **CGPA Calculator** based on the **Amrita University grading system**.
This tool helps students estimate their **CGPA by entering expected grades and subject credits**.

---

## Features

* Add multiple subjects
* Enter **credits for each subject**
* Select **expected grades**
* Default **grade point system included**
* Editable **grade point table**
* Automatic **CGPA calculation**
* Displays:

  * Total Credits
  * Total Grade Points
  * Final CGPA
* Reset option to clear all inputs
* Simple and clean student-friendly interface

---

## Grading System (Default)

| Grade | Grade Point |
| ----- | ----------- |
| O     | 10          |
| A+    | 9.5         |
| A     | 9           |
| B+    | 8           |
| B     | 7           |
| C     | 6           |
| P     | 5           |
| F     | 0           |

These values can be **modified by the user if needed**.

---

## CGPA Formula

CGPA is calculated using the formula:

CGPA = Σ (Credits × Grade Point) / Σ (Credits)

Where:

* **Credits** → credit value of the subject
* **Grade Point** → numerical value corresponding to the grade

---

## Example

| Subject     | Credits | Grade |
| ----------- | ------- | ----- |
| Mathematics | 4       | A     |
| Physics     | 3       | B+    |
| Programming | 4       | O     |

The calculator multiplies each **credit with its grade point**, sums them, and divides by **total credits** to compute CGPA.

---

## How to Use

1. Add your subjects.
2. Enter **credits** for each subject.
3. Select your **expected grades**.
4. The calculator will automatically compute your **CGPA**.
